Sinkron Shopee

Memeriksa status koneksi...

Hubungkan toko Shopee Anda lewat Shopee Open Platform untuk menarik & mendorong data stok secara otomatis. Produk dicocokkan berdasarkan SKU — pastikan SKU di sistem ini sama dengan SKU di Shopee Seller Center.

Cara kerja singkat

  1. Daftar App di open.shopee.com, dapatkan Partner ID & Partner Key.
  2. Isi kredensial itu ke file .env di server (lihat README).
  3. Klik "Hubungkan Toko Shopee" di atas, login dengan akun Seller Center Anda.
  4. Setelah terhubung, klik "Tarik Stok dari Shopee" untuk sinkron awal berdasarkan SKU.
  5. Selanjutnya, setiap perubahan stok manual bisa didorong ke Shopee lewat tombol "Kirim ke Shopee" di halaman Produk.
